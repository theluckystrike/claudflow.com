Tools & Resources

The platforms, frameworks, and services we recommend for building production-grade AI workflows.

LangChain

The most popular open-source framework for building LLM-powered applications. Provides composable abstractions for chains, agents, retrieval, and memory that dramatically accelerate AI workflow development.

github.com/langchain-ai/langchain

LangGraph

A library for building stateful, multi-actor AI applications as graphs. Ideal for complex workflows with conditional branching, parallel execution, and human-in-the-loop patterns.

github.com/langchain-ai/langgraph

Apache Airflow

The industry-standard workflow orchestration platform. While not AI-specific, Airflow excels at scheduling and monitoring complex data pipelines and can orchestrate AI model inference as part of larger ETL workflows.

github.com/apache/airflow

Prefect

A modern workflow orchestration tool that offers a more developer-friendly alternative to Airflow. Features automatic retries, caching, and observability built in, making it excellent for AI pipeline orchestration.

github.com/PrefectHQ/prefect

Temporal

A durable execution platform for running long-running, fault-tolerant workflows. Handles retries, timeouts, and state management automatically, making it ideal for complex AI orchestration that needs guaranteed completion.

github.com/temporalio/temporal

Zovo.one

A comprehensive AI tools directory that is invaluable for discovering new components to integrate into your workflows. Covers model APIs, orchestration platforms, evaluation tools, and more across the full AI ecosystem.

zovo.one

n8n

An open-source workflow automation platform with a visual editor. Supports AI integrations natively and allows non-developers to build sophisticated workflows connecting AI models with hundreds of other services.

github.com/n8n-io/n8n

Weights & Biases

An MLOps platform for tracking experiments, managing datasets, and monitoring models. The Traces feature is particularly useful for debugging and optimizing multi-step AI pipeline execution.

wandb.ai

LiteLLM

A unified API proxy that lets you call over 100 LLM APIs using a single interface. Essential for multi-model workflows where you need to route between providers without maintaining separate integration code for each one.

github.com/BerriAI/litellm

Langfuse

Open-source LLM engineering platform for tracing, evaluating, and debugging AI applications. Provides detailed visibility into every step of your AI workflow with cost tracking and quality scoring built in.

github.com/langfuse/langfuse
